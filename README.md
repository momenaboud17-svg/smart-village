# Smart Village - القرية الذكية

## 🏥 مشروع إنساني تقني ثوري
### Created by Momen Aboud

تطبيق طبي ريفي متكامل يعمل بدون إنترنت، يهدف لخدمة كبار السن والأميين في القرى.

## ✨ المميزات الرئيسية

1. **البوابة العصبية (Neural Gateway)**
   - ماسح ضوئي حيوي للعلامات الحيوية
   - تحليل لوني للعين والجلد
   - قياس النبض والأكسجين والحرارة

2. **مصفوفة التشخيص (Diagnostic Matrix)**
   - محرك استنتاج ذكي (Expert System)
   - تشخيص: أنيميا، حساسية صدر، التهاب رئوي، جلطات
   - بروتوكولات طوارئ فورية

3. **النبي الصوتي (Vocal Prophet)**
   - قراءة التشخيص بصوت واثق بلهجة ريفية
   - يعمل بدون إنترنت (Offline TTS)

4. **مستمع اللهجات (Dialect Listener)**
   - يفهم العامية الريفية: "صدري بيزيّق"، "دماغي هتنفجر"
   - يحولها لبيانات طبية دقيقة

5. **خريطة القرية الحرارية**
   - تعمل بالبلوتوث (Peer-to-Peer)
   - تنبه لانتشار الأمراض في الجوار

6. **لوحة تحكم مومن**
   - إحصائيات الأرواح التي تم حمايتها
   - توقيع متوهج "Created by Momen Aboud"

## 📱 التثبيت

1. افتح الموقع في المتصفح
2. اختر "إضافة إلى الشاشة الرئيسية"
3. استمتع بالتطبيق بدون إنترنت

## 🛠 التقنيات المستخدمة

- HTML5 / CSS3 / JavaScript (ES6)
- PWA (Progressive Web App)
- Web Speech API (للنطق)
- LocalStorage (للبيانات)
- Peer-to-Peer محاكاة (للخريطة)
